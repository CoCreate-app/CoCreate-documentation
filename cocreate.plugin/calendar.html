Attributes

  • data-btn_type string optional

    "timeGridDay" / "timeGridWeek" / "dayGridMonth" / "today"

  • data-calendar_id string optional

    Calendar id

  • data-dispaly_field string optional

    Display field

  • data-pass_document_id string optional

    Pass Document id
