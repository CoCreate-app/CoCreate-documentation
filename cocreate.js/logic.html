Passing Attributes

Attributes

  • data-pass_to string required

    Save data onChange. If attribute is empty or not used Default's to true.

  • data-pass_id string required

    Save data onChange. Default's to true if not appllied.

  • data-pass_collection string optional

    Save data onChange. Default's to true if not appllied.

  • data-pass_document_id string optional

    Save data onChange. If attribute is empty or not used Default's to true.

  • data-pass_prefix string optional

    Fetch value on page load or when updated. If attribute is empty or not used Default's to true.

Reference

Passing attributes is how you will create a flow.

For example you may have a customer profile that when you click on the edit icon a form opens for editing the customers information.

For passing attributes we utilize <a> tag.

Using </a> tag you can open a modal, movable window/modal or new page in the same tab, new tab or new window.

And pass an id to insert into that page's html to dynamically display data in realtime.
